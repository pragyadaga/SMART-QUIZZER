In combined_ltk.py
	modify the path to pdftotext.exe

Put pywin32 in the default python directory

run as 
	python combined_nltk.py <filename.pdf>
	
	Eg. python combined_nltk.py C.pdf

if .pdf and only a part of the document is needed then run as
	python file_convert_read.py <input_file_name.pdf> start_page_no last_page_no

	Eg. python combined_nltk.py input.pdf 30 54


Requirements:
	Windows OS
	python34
	nltk package
